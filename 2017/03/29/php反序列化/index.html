php中的序列化和反序列化

把复杂的数据类型压缩到一个字符串中
serialize() 把变量和它们的值编码成文本形式
unserialize() 恢复原先变量

eg：

$m = array('Moe','Larry','Curly');
$n = serialize($m);
print_r($n);echo "<br />";
print_r(unserialize($n));

结果：a:3:{i:0;s:3:”Moe”;i:1;s:5:”Larry”;i:2;s:5:”Curly”;}
Array ( [0] => Moe [1] => Larry [2] => Curly )
